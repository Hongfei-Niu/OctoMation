# Octomation编排自动化产品应用能力APP

# 使用方法

下载您需要的应用APP安装包，访问OctoMation应用管理界面https://<OCTOMATION_SERVER>/apps，点击【上传应用】。

部分APP上传后可能需要配置资源参数，如：IP、端口、账号、密码、API KEY等。


![上传APP](./octomation-app-upload.png)
