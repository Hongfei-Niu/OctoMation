# OctoMation剧本包

# 使用方法

下载您需要的剧本包，访问OctoMation的剧本管理界面https://<OCTOMATION_SERVER>/playbook_list，点击【导入】按钮。

剧本上传后，需要您手工编辑一次剧本内容，将相关的应用及参数设置为本地环境，然后再以运行。

![上传剧本](./octomation-playbook-upload.png)
